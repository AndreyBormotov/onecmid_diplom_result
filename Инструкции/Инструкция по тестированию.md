### Инструкция для проведения автоматического тестирования подсистемы обслуживание клиента"

1. Необходимо создать новую пустую базу, через конфигуратор добавить пользователя Администратор с полными правами.
2. Скачайте Vanessa Automation из [официального гит-репозитория](https://pr-mex.github.io/vanessa-automation/dev/).
3. Распакуйте скачанный архив Vanessa Automation в корень диска (например C:\vanessa-automation\)
4. Для запуска Vanessa Automation в режиме менеджера тестирования необходимо запустить информационную базу с параметрами: 
    /TESTMANAGER /N "Администратор" /Execute С:\vanessa-automation\vanessa-automation.epf /C "StartFeaturePlayer"
5. Нажимаем команду открыть основную форму, затем загрузить -> загрузить файлы из каталога и выбираем папку с предоставленными сценариями тестирования
6. Запустится последовательное выполнение сценариев тестирования
7. По окончанию процесса выполнения сценариев тестирования снизу появится сообщение об ошибках при выполнении сценарного тестирования или об успешном завершении всех тестов.
