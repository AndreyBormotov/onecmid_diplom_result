### Инструкция для подготовки работы с новой подсистемой "Работа с заявками клиентов"

1. Обновить конфигурацию из файла (.cf) с помощью конфигуратора через сравнение/объединение приняв все изменения.
2. Запустить информационную базу с параметром (/C ЗапуститьОбновлениеИнформационнойБазы).
3. Назначить новые профили доступа (Специалист, Менеджер, Бухгалтер ИТ Фирмы, Кадровик-Расчетчик) пользователям ИБ в соответствии с компетенциями.
4. Заполнить справочник Зарплата -> Графики работы необходимыми графиками
5. Заполнить регистр сведений графики работ обработкой Зарплата -> Заполнение графика
6. Заполнить справочник Зарплата -> Сотрудники, обязательно указав категорию и условия оплаты. (Ник в телеграмм указывать необязательно, актуально для специалистов для упоминания ботом при отправке сообщения в группу).
7. Создать в справочнике Покупки и продажи -> Номенклатура две номенклатуры "Абонентская плата" и "Работа специалиста" 
8. Заполнить в форме настроек Обслуживание клиентов -> Настройки созданную номенклатуру, а также значения для интеграции с телеграм ботом ([Инструкция для получения токена бота и Id группы в Телеграм](НастройкаТелеграм.md))
9. Для начала использования новой подсистемы в работе необходимо создать договоры с контрагентами с видом "Абонентское обслуживание" обязательно заполнив все необходимые реквизиты.
