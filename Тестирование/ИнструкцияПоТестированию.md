### Инструкция для запуска автоматического тестирования корректности работы новой подсистемы "Работа с заявками клиентов"

1. Обновить конфигурацию из файла (.cf) с помощью конфигуратора через сравнение/объединение приняв все изменения.
2. Запустить информационную базу с параметром (/C ЗапуститьОбновлениеИнформационнойБазы).
3. Назначить новые профили доступа (Специалист, Менеджер, Бухгалтер ИТ Фирмы, Кадровик-Расчетчик) пользователям в соответствии с компетенциями.
4. Заполнить справочник сотрудники, обязательно указав категорию и условия оплаты. (Ник в телеграмм указывать необязательно, актуально для специалистов для упоминания ботом в сообщении). 
5. Заполнить форму настроек Обслуживание клиентов -> Настройки ([Инструкция для получения токена бота и Id группы в Телеграм](НастройкаТелеграм.md))
